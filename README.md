# vhr-remake 微人事重制版

## 修改

### 数据库

* **表字段名**从驼峰改为**下划线**

* 部分字段改为**不能为null**

* 删除**中间表ID**

### 前端

* **Webpack**改为**Vite**
* **JavaScript**改为**Typescript**
* **Vue2**改为**Vue3**
* **Element UI**改为**Element Plus**

### 后端

* **Mybatis**改为**Jimmer**
* **SpringBoot2**改为**SpringBoot3**
* **SpringSecurity**改为**SaToken**
*  优化导入导出**Excel**代码

### 整体

- 改为**前后端分离**
